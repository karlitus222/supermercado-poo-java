# Sistema de Supermercado

Projeto desenvolvido para a disciplina de Programação Orientada a Objetos.

## Estrutura do Projeto
- Produto
- Perecivel
- ProdutoImportado
- Venda
- Classe principal: Supermercado

## Conceitos Aplicados
- Encapsulamento
- Herança
- Polimorfismo

## Como Executar
1. Abra o projeto em uma IDE (NetBeans ou IntelliJ).
2. Execute a classe `Supermercado.java`.
